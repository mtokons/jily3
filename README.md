# Jily Enterprise Cross-Platform Sales Entry App

## Project Overview
This app allows Jily Enterprise to manage daily sales entries. Admins can add products and prices. Sales representatives can record daily product sales quantities. The app will be available for both web and Android platforms.

## Tech Stack
- Web: React
- Android: React Native
- Backend: Node.js (Express)
- Database: SQLite

## Features
- Admin: Add/edit products and prices
- Sales Rep: Enter daily sales (product, quantity)

## Getting Started
1. Install dependencies: `npm install`
2. Start backend: `npm run server`
3. Start web app: `npm start`
4. For Android: Use React Native CLI or Expo

## Folder Structure
- `/backend` - Node.js API & SQLite
- `/web` - React web app
- `/mobile` - React Native app
